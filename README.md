# phillips-growlers

`phillips-growlers` is a small CLI tool to print the current growler offerings
available from Phillips Brewery.

### Installation

    $ go get -u github.com/stewart/phillips-growlers

### Usage

    $ phillips-growlers
    - Analogue 78 Kolsch
    - Blue Buck Ale
    - Hop Circle IPA
    - Longboat Chocolate Porter
    - Mantra Saison
    - Pilsner Pilsner
    - Raspberry Wheat Ale
    - Short Wave Pale Ale
